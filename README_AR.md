# Crypto Signals (بدون سيرفر) — Kotlin / Jetpack Compose

مشروع تطبيق توصيات يعمل بالكامل بدون أي Backend خاص بك:
- يجلب الأسعار من Binance مباشرة (REST)
- يولّد توصيات داخل التطبيق (تحليل فني + فلاتر)
- إشعارات محلية (بدون Firebase/FCM)
- حفظ محلي Room + Dashboard
- Backtest بسيط داخل التطبيق (MVP)

## مهم جدًا (إخلاء مسؤولية)
التطبيق لأغراض تعليمية/معلوماتية فقط، وليس نصيحة مالية.
التداول ينطوي على مخاطر وقد تخسر رأس مالك. لا يوجد نظام يضمن أرباح.

## تشغيل المشروع
1) افتح المجلد في Android Studio
2) Sync Gradle
3) Run على جهازك

## كيف يعمل
- WorkManager دوري (كل 15 دقيقة) يفحص أعلى عملات USDT سيولة ويولد إشارات "قليل لكن قوي"
- Monitor Worker يتحقق من TP/SL ويرسل إشعارات الربح/الخسارة (Best-effort حسب قيود Android)

## تخصيص سريع
من شاشة الإعدادات:
- رأس المال لكل توصية (USDT)
- الحد الأدنى للـ Score
- الوضع الداكن
